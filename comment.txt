Дата: 4/6/2026 (Ришат)
Основные изменения:
    Убраны шаги ручного ввода:
        имени
        фамилии
        института
        группы
    Регистрация теперь проходит по следующему сценарию:
        Пользователь вводит логин КНИТУ ONE
        Вводит пароль
        Бот проверяет данные через API
        Автоматически получает:
        имя и фамилию
        номер группы
        институт (определяется по группе)
    Добавлена автоматическая валидация данных:
        при неверном логине или пароле регистрация не продолжается
    Перед завершением регистрации пользователь:
        принимает политику обработки персональных данных
        подтверждает корректность полученных данных
    Дополнительно:
        Удалены лишние состояния и обработчики
        Повышена точность и надежность данных
        Улучшен пользовательский опыт (меньше действий, быстрее регистрация)
        Добавлена отправка стикеров при регистрации
        Был разработан сайт-визитка бота
